This Telegram bot fetches Chuck Norris jokes from a web page and allows users to set
their preferred language to receive jokes in the chosen language.

connecting to the bot-
1. Search for @chucknorrisjokebot on telegram, or use this link- https://t.me/EinavK_Bot.
2. Click on start to start the bot.
3. Set your preferred language by using the command '/set language <language>'-
    for example- /set language hebrew
once the language is set, send a number between 1 and 101 to receive a joke in the selected language.
4. when using the '/help' command,  you will recive instractions from the bot.
